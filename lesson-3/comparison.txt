=== Docker Image Comparison ===

1. Fat Image: fat-model
----------------------------------------
Size: 1.86 GB
Number of layers: 18
- Основні шари:
  - Python + system deps
  - pip install torch, torchvision, pillow
  - Копіювання inference.py та model.pt
  - CMD ["python" "inference.py"]

2. Slim Image: slim-model-min
----------------------------------------
Size: 830 MB
Number of layers: 16
- Основні шари:
 - COPY /usr/local/lib/python3.9/site-packages /usr/
 - RUN /bin/sh -c set -eux; savedAptMark="$...

3. Рекомендації щодо подальшої оптимізації slim:
- Використовувати офіційні CPU-only wheels для PyTorch і torchvision, щоб уникнути встановлення зайвих бібліотек CUDA.
- Видаляти зайві пакети після встановлення та обмежувати тільки runtime dependencies.
- Об'єднати RUN-команди для мінімізації кількості шарів.
- Перевірити можливість використання python:slim або python:alpine як базового образу для подальшого зменшення розміру.

=== Висновки ===
- Slim images значно легші (768MB/830MB vs 1.86GB)
- Обидва образи виконують inference коректно
